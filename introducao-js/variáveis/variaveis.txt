VARIÁVEIS

1. Servem para armazenar os dados para podermos usá-los no nosso código.

2. Criação de variáveis: var, let & const 

var - não é usado mais nos navegadores mais modernos.
let - forma mais moderna de trabalhar com variáveis.
const - 'variávies' de valores fixos. 


------ console ------
Ex.: var nome = "quirino"

< undefined 

nome
< 'quirino'

------ console ------

let dia = 1

< undefined 

dia
< 1

------ console ------

const mes = 'julho'

< undefined 

mes

< 'julho'

mes = 'agosto'
< variável já declarada e não pode ser modifcado

NOMEAÇÃO DE VARIÁVEIS

Nomes significativos para as variáveis - ligações

Ex.: let nome = 'quirino' (forma correta)

Ex.: let x = 'quirino' (forma incorreta)

obs.: Não é possível declarar duas variáveis com o mesmo nome.

Camel Case - formas

let nomeDoUsuario = 'quirino'
let notaDoUsuario = 10

Snake Case - formas

let nome_do_usuario = 'quirino'

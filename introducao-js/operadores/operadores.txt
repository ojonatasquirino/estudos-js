OPERADORES & EXPRESSÕES

ARITMÉTICOS -> + - * / %

Operadores e modo de prevalência na matemática (por agrupamento).

+ soma
- subtração
* multiplicação
/ divisão
% resto da divisão

ATRIBUIÇÃO -> = += -= *= /= 

= 

Ex.: let nome = 'quirino'
---> o valor de nome é quirino

+= (soma)

------ console ------


Ex.: let a = 5
a += 5
< 10

---> acrescenta o valor ao número existente

------ console ------


-= (subtração)

Ex.: let a = 5
a -= 2
< 3

o mesmo vale para os outros operadores.

CONCATENAÇÃO

Obs.: String + String = String
Number + String = String
Number + Number = Number
